# 23. Introducción al Cracking de Contraseñas

En esta sección vamos a estar viendo **cracking de contraseñas**. Qué ataques podemos llevar a cabo para adivinar contraseñas y sacar
contraseñas.

Vamos a empezar definiendo que es un **ataque de fuerza bruta**, que se usa mucho para el cracking de contraseñas.

## Ataque de fuerza bruta

Es un intento de descifrar una contraseña o nombre de usuario, que consiste en aplicar el método de prueba y error, de probar contraseñas
hasta dar con una correcta.

Probar contraseñas, probar usuarios, porque no solamente la fuerza bruta ocurre con contraseñas, ocurre también con usuarios. Pero normalmente se
hace con contraseñas.

Una, que se tiene el usuario por medio de hacer una enumeración.

Podemos, por medio de usar herramientas, conseguimos una lista de posibles usuarios.

Podemos probar un ataque de fuerza bruta a un servicio, a un login.

Podemos en una página de login, de una página web, podemos probar lo que vendría siendo estos ataques de fuerza bruta para poder descifrar la
contraseña.
