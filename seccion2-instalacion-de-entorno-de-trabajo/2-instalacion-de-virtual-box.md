# 2. Instalación de Virtual Box

**VirtualBox** es una aplicación la cual nos sirve para poder crear máquinas virtuales.

Una **máquina virtual** es una máquina, es una computadora, la cual corre en nuestra computadora principal.

* Link de descarga de [VirtualBox](https://www.virtualbox.org/wiki/Downloads)

Elegimos según el sistema que estemos utilizando, que en nuestro caso, es Windows. Y descargamos el ejecutable.

Vamos a darle un clic derecho y **Ejecutar como administrador**.

* **NOTA**: La mayoría de las opciones elegir siguiente, siguiente, siguiente.

Y ya tenemos la aplicación de VirtualBox instalada.
